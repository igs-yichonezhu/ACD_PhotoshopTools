### 更新內容

#### 新功能
- **更新通知 Release Notes** — 有新版本時可展開查看更新了什麼
- **共用資源工具** — 新增「共用資源」分類，收錄公司內部常用連結（商用美術議題、內部工具、AI 帳號、opotool、ACDPik 算圖）
- **Usage Log** — 記錄工具使用情況（本地 log + Google Sheets）
- **連線狀態顯示** — 設定頁面顯示目前連線模式

#### 介面優化
- **Freepik 風格卡片** — 彩色漸層縮圖 + 左上角標籤 + 下方文字
- **現代深色主題** — 更柔和的色調、更大的圓角、膠囊形按鈕
- **兩欄排列** — 卡片更大更易讀
- **面板尺寸加大** — 預設 420×750

#### 修正
- **支援 Photoshop 2025-2027** — 新增 CSXS 14-15 Debug Mode 設定
- **Token 改為選填** — 公開 Repo 不需要 GitHub 帳號或 Token
- **安裝程式預填 Repository** — 安裝時不用手動輸入

#### 移除
- 移除「快速取色」工具

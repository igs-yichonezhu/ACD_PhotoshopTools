## 2. 取得安裝檔與設定資料

### 安裝檔

到 GitHub Releases 頁面下載最新版本的安裝檔：

> https://github.com/igs-yichonezhu/ACD_PhotoshopTools/releases

下載 `igs-arttools-setup-vX.X.X.exe`（例如 `igs-arttools-setup-v1.3.0.exe`）。

### GitHub 設定資料

安裝時需要填入以下兩項，用於自動更新功能：

| 項目 | 值 |
|------|------|
| **Repository 名稱** | `igs-yichonezhu/ACD_PhotoshopTools` |
| **GitHub Token** | 請依照下方步驟產生 |

### 如何產生 GitHub Token（PAT）

1. 打開 https://github.com/settings/tokens?type=beta
2. 點擊 **Generate new token**
3. 填寫設定：
   - **Token name**：`ACD-ArtTools`（名稱隨意）
   - **Expiration**：建議選 `90 days` 或更久
   - **Repository access**：選 **Only select repositories** → 勾選 `ACD_PhotoshopTools`
   - **Permissions** → 展開 **Repository permissions** → 找到 **Contents** → 設為 **Read-only**
4. 點擊 **Generate token**
5. **立即複製**顯示的 `github_pat_...` 字串（離開頁面後就無法再看到）

> ⚠️ Token 請妥善保管，不要外傳。如果 Token 過期，重新依照上述步驟產生一組新的即可。

## 3. 安裝步驟

### 步驟 1：關閉 Photoshop

安裝前請先**完全關閉** Photoshop。

### 步驟 2：執行安裝檔

雙擊 `igs-arttools-setup-vX.X.X.exe`，依照畫面指示操作：

1. **歡迎頁面** → 點擊「下一步」
2. **安裝路徑** → 保持預設值，點擊「下一步」
   - 預設路徑：`%APPDATA%\Adobe\CEP\extensions\com.igs.arttools`
3. **GitHub 設定** → 填入管理員提供的資訊：
   - **GitHub Personal Access Token (PAT)**：貼上 `github_pat_...` 那串文字
   - **GitHub Repository**：填入 `igs-yichonezhu/ACD_PhotoshopTools`
   - > 這兩項也可以稍後在插件設定中修改，此步驟可先留空
4. **安裝中** → 等待安裝完成
5. **完成** → 點擊「完成」

### 步驟 3：啟動 Photoshop

開啟 Photoshop，在選單列找到：

```
視窗 (Window) → 延伸功能 (Extensions) → IGS ACD Art Tools
```

點擊後即可開啟工具面板。

## 4. 首次啟動與設定

### 如果安裝時已填入 GitHub 資訊

插件會直接進入主畫面，顯示所有可用工具。完成！

### 如果安裝時跳過 GitHub 設定

首次開啟插件時會自動彈出設定視窗：

1. **GitHub Personal Access Token** — 貼上管理員提供的 Token
2. **GitHub Repository** — 填入 `igs-yichonezhu/ACD_PhotoshopTools`
3. 點擊「**儲存**」

之後若需要修改設定，點擊面板右上角的 **⚙ 齒輪圖示** 即可。

## 5. 更新方式

### 自動更新（建議）

每次開啟 Photoshop 時，插件會自動檢查是否有新版本。

如果偵測到新版本，面板頂部會出現提示：

```
┌───────────────────────────────────┐
│  新版本可用 v1.2.0       [ 更新 ] │
└───────────────────────────────────┘
```

點擊「**更新**」按鈕，插件會自動：
1. 下載新版本
2. 安裝更新
3. 提示重啟 Photoshop

**重啟 Photoshop 後即完成更新。**

### 手動檢查更新

點擊面板右上角的 **↻ 重新整理按鈕**，可手動檢查是否有新版本。

## 6. 解除安裝

1. 開啟 Windows **設定** → **應用程式** → **已安裝的應用程式**
2. 搜尋「**IGS ACD Art Tools**」
3. 點擊「**解除安裝**」
4. 安裝程式會詢問是否同時移除設定檔（GitHub Token 等）
   - 選「**是**」= 完全移除
   - 選「**否**」= 保留設定，下次重新安裝時不用再填

## 7. 常見問題

### Q：在 Photoshop 找不到插件？

**A：** 請確認以下幾點：
1. Photoshop 版本為 CC 2019 或更新
2. 到 `視窗 → 延伸功能` 查看（不是「視窗 → 延伸功能（舊版）」）
3. 安裝後是否有重啟 Photoshop
4. 嘗試重新執行安裝檔

---

### Q：開啟插件後面板是空白的？

**A：** 可能是 CEP Debug Mode 沒有正確啟用。請：
1. 重新執行安裝檔（安裝程式會自動設定）
2. 重啟 Photoshop

---

### Q：按了「更新」但失敗？

**A：** 請確認：
1. 網路連線正常
2. GitHub Token 是否正確（點 ⚙ 齒輪圖示檢查）
3. Token 是否過期（過期需要向管理員重新取得）
4. 如果持續失敗，可直接向管理員索取新版 `.exe` 重新安裝

---

### Q：Token 過期了怎麼辦？

**A：** 向管理員索取新的 Token，然後：
1. 開啟插件面板
2. 點擊右上角 ⚙ 齒輪圖示
3. 貼上新的 Token
4. 點擊「儲存」

---

### Q：我需要 GitHub 帳號嗎？

**A：** **不需要。** 你只需要管理員提供的 Token 和 Repository 名稱，不需要自己的 GitHub 帳號。

---

### Q：安裝後 Photoshop 會變慢嗎？

**A：** 不會。插件只在你開啟面板時才運作，不影響 Photoshop 其他功能。

---

### Q：可以同時安裝在多台電腦嗎？

**A：** 可以。每台電腦各自安裝一次 `.exe`，使用同一組 Token 和 Repository 即可。

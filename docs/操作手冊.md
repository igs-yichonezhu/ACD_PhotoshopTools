## 1. 安裝與啟動

### 安裝位置

將 `com.igs.arttools` 資料夾放入：

```
Windows:  %APPDATA%\Adobe\CEP\extensions\com.igs.arttools\
macOS:    ~/Library/Application Support/Adobe/CEP/extensions/com.igs.arttools/
```

### 啟用 Debug 模式（開發時需要）

Windows 登錄編輯程式 (regedit)，新增或修改：

```
HKEY_CURRENT_USER\SOFTWARE\Adobe\CSXS.9
PlayerDebugMode = "1"  (字串值)
```

### 開啟面板

Photoshop 選單 → **視窗(W)** → **延伸功能(舊版)** → **IGS ACD Art Tools**

## 2. 主面板總覽

面板啟動後顯示工具卡片網格，提供以下功能：

| 區域 | 說明 |
|------|------|
| 類別篩選列 | 全部 / 影像 / 文字 / 色彩 / 匯出 / 工作流程 / 圖層 / UI |
| 工具卡片 | 點擊進入對應工具 |
| ⚙ 設定 | GitHub Token 與 Repo 設定（自動更新用） |
| 底部狀態列 | 已載入工具數量、版本號 |
| 更新橫幅 | 有新版本時自動顯示 |

點擊任一工具卡片後會進入該工具頁面，左上角 **← 返回** 可回到主面板。

## 3. 工具一：快速取色

**用途**：快速讀取 / 設定 Photoshop 前景色與背景色。

### 操作步驟

1. 開啟工具後自動讀取目前前景色 / 背景色
2. **前景色區域**：顯示色塊 + RGB 值 + HEX 值
3. **背景色區域**：同上
4. **輸入 HEX 值**：在輸入框填入色碼（如 `#FF5500`），點擊「設定」套用
5. **交換**：前景 ↔ 背景色互換
6. **重設**：恢復為預設黑白
7. **重新整理**：從 Photoshop 重新讀取色彩

## 4. 工具二：文件資訊

**用途**：顯示當前文件的完整資訊。

### 顯示內容

- 文件名稱、儲存狀態
- 畫布尺寸（寬 × 高）
- 解析度 (DPI)
- 色彩模式（RGB / CMYK / etc.）
- 圖層數量
- 圖層列表（含樹狀結構、可見性 👁、透明度 %）

點擊 **重新整理** 可更新資訊。

## 5. 工具三：特效去背

**用途**：將黑底（或其他純色底）特效圖層轉換為透明背景 PNG。適合匯出粒子特效、光效等素材。

### 操作步驟

1. 在 Photoshop 中選擇特效圖層
2. 開啟本工具
3. **控制 Tab**：
   - 選擇底色（黑 / 白 / 自訂 HEX）
   - 可用滴管工具取色
   - 調整閾值 (Threshold, 0~100)：控制邊緣柔和度
4. **預覽 Tab**：即時預覽去背效果（棋盤格背景）
5. 滿意後點擊 **匯出 PNG** 儲存透明背景圖檔

## 6. 工具四：UI 匯出到 Cocos

**用途**：將 PSD 的 UI 版面匯出為 Cocos Creator 可用的 `layout.json` + 圖片資源。

### 前置準備

- 建議文件 DPI 設為 **72**（非 72 會顯示警告但不阻止匯出）
- 確認圖層命名有意義（圖層名會成為 Cocos 節點名）

### 操作步驟

#### Step 1：設定

| 欄位 | 說明 |
|------|------|
| 匯出路徑 | Cocos 專案的 `assets` 資料夾路徑 |
| 子資料夾 | 匯出目標子資料夾，預設 `ui-import` |
| 縮放比例 | 1 = 原尺寸，0.5 = 縮小一半 |
| 文字字型 | Arial (Cocos 預設) 或 當前 PSD 字型 |

#### Step 2：檢查

點擊 **「檢查」** 按鈕，會：
1. 讀取 PSD 完整圖層樹
2. 彈出 **圖層限制報告**，列出：
   - ⊘ **調整圖層**（無法匯出，自動跳過）
   - ✦ **圖層效果 FX**（文字 Label 不帶效果，可手動切換為圖片匯出）
   - ⬇ **剪裁遮色片**（遮罩關係會遺失）
   - ◆ **智慧型物件**（匯出為平面化 PNG）
   - ◐ **混合模式**（Cocos 僅支援 Normal）
   - S̶ **刪除線文字**（cc.Label 不支援）

#### Step 3：選擇圖層

- 勾選要匯出的圖層（預設全選）
- 每個文字圖層旁有 **[T] / [🖼]** 切換按鈕：
  - **T**（預設）= 匯出為 Cocos Label 文字節點
  - **🖼** = 匯出為圖片 Sprite（適合有特效的文字）
- 圖層標籤顏色：🟢 圖片、🟠 文字、🔵 群組
- 紫色 **FX** 標籤表示該圖層有圖層效果

#### Step 4：匯出

點擊 **「匯出到 Cocos」**：
- 進度條顯示匯出進度
- 可隨時按 **STOP** 停止
- 完成後顯示匯出統計（圖片數、文字數）

### 匯出產物

```
assets/ui-import/
├── layout.json          ← 版面結構資料
└── images/
    ├── 背景.png
    ├── 按鈕_1.png
    └── ...
```

### 文字處理特殊邏輯

| 情境 | 處理方式 |
|------|---------|
| 一般文字 | 匯出為 Label，帶字型、大小、顏色、對齊 |
| 有 Free Transform 的文字 | 自動計算視覺字體大小（base × transform） |
| 有 FX 效果的文字 | 預設匯出 Label（可手動切為圖片） |
| 直排文字 | 自動偵測，設定 overflow=RESIZE_HEIGHT |
| 混合字型大小 | DOM 失敗時自動用 Action Manager 讀取第一段樣式 |

## 7. Cocos Creator 端：匯入 PSD UI Layout

### 安裝 Cocos 擴充套件

將 `igs-ui-importer` 資料夾放入 Cocos 專案的 `extensions/` 目錄：

```
your-project/
├── assets/
├── extensions/
│   └── igs-ui-importer/
│       ├── package.json
│       └── dist/
│           ├── main.js
│           └── scene/
│               └── build-node-tree.js
└── ...
```

重啟 Cocos Creator 使擴充套件生效。

### 操作步驟

1. 確認 `layout.json` 和 `images/` 已在 `assets/ui-import/` 下
2. 等待 Cocos 匯入圖片資源完成（Asset Database 同步）
3. Cocos 選單 → **IGS Tools** → **匯入 PSD UI Layout**
4. 選擇 `layout.json` 檔案
5. 自動生成 `.prefab` 檔案

### 生成的 Prefab 結構

```
PSD_文件名 (Root Node)
├── 背景 (Sprite)
├── middle_general (Group)
│   ├── 狂歡派對 (Label)
│   ├── 老虎機 (Label)
│   └── ...
├── option2_展 2 (Group)
│   ├── 聯絡我們 (Label)
│   └── ...
└── ...
```

### 支援的 Cocos 元件

| PSD 圖層類型 | Cocos 元件 |
|-------------|-----------|
| 一般圖層 / 智慧物件 | Node + Sprite + UITransform |
| 文字圖層 | Node + Label + UITransform |
| 群組 | Node + UITransform（容器） |
| 透明度 < 100% | UIOpacity 元件 |

### 完成對話框

匯入完成後會顯示：
- 成功統計（圖片 / 文字 / 群組數量）
- 警告列表（直排文字、FX 文字等需手動調整項目）
- 失敗的 Sprite（找不到對應圖片的圖層）

## 8. 自動更新

### 設定

1. 主面板右上角 **⚙** → 設定
2. 填入 **GitHub Token**（Personal Access Token，需 `repo` 權限）
3. 填入 **Repository**（格式：`owner/repo`）
4. 點擊「儲存」

### 更新流程

1. 啟動面板時自動背景檢查（2 秒延遲）
2. 有新版本時頂部顯示更新橫幅
3. 點擊「更新」按鈕
4. 自動下載 → 解壓 → 替換檔案
5. 提示重啟 Photoshop

## 9. 常見問題 FAQ

### Q：面板打開後是空白的？
**A**：確認 Debug 模式已啟用（regedit 設定 `PlayerDebugMode = "1"`），然後重啟 Photoshop。

### Q：DPI 不是 72 會怎樣？
**A**：會顯示警告提醒你調整，但不會阻止匯出。建議在 PS 中調整：**影像 → 影像尺寸 → 解析度改為 72**（取消勾選「重新取樣」避免改變畫布像素尺寸）。

### Q：文字大小跟 PS 不一致？
**A**：如果文字圖層有被 Free Transform 縮放過，工具會自動讀取 transform 矩陣計算視覺字體大小。若仍不一致，請確認 PS 字元面板上顯示的實際數值。

### Q：背景圖層匯出失敗？
**A**：PS 的鎖定背景圖層（Background layer）匯出時會自動轉換處理。若仍失敗，可手動在 PS 中雙擊背景圖層將其轉為一般圖層。

### Q：Cocos 匯入後看不到 Sprite 圖片？
**A**：確認圖片已被 Cocos Asset Database 匯入完成（等進度條跑完），然後再執行「匯入 PSD UI Layout」。

### Q：匯出很慢？
**A**：每個圖層匯出 PNG 時需要 PS 進行 duplicate + flatten + save，大量圖層會較慢。可以取消勾選不需要的圖層來加速。匯出過程中可隨時按 STOP 中斷。

### Q：文字有圖層效果（陰影、描邊等）怎麼辦？
**A**：Cocos 的 Label 元件不支援圖層效果。你可以：
1. 保持 [T] 模式，匯出純文字 Label，在 Cocos 端手動加效果
2. 切換為 [🖼] 模式，將文字連同效果一起匯出為圖片
